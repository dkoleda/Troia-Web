Troia is service that runs within tomcat server and takes requests from via HTTP. It uses POST and GET methods to send request data. Service address --------------- To call services provided by Troia you have to call HTTP connection to correct location. Correct format is as follows :: http://://rest/ Where: - is address of host providing Troia - is port on with service is available - is path in with Troia files are located on server - is name of service that's going to be called
